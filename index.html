Abstände

Abstände innerhalb des Elements

padding (top, left, bottom, right)

Erzwungener Leerraum zwischen dem Inhalt eines Elements und seinem eigenen Elementrand, also z.B. zwischen dem Text eines Elements und dem Rand dieses Elements.

Abstände außerhalb des Elements

margin (top, left, bottom, right)

Erzwungener Leerraum zwischen dem aktuellen Element und seinem Eltern- bzw. seinen Nachbarelementen.

Höhe und Breite eines Elements

(max-)width / (max-)height

Mit den width oder height properties kann eine (maximale) Höhe oder Breite eines Elements festgelegt werden.
