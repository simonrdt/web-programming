Flexbox & CSS Grid

Welche Möglichkeiten gibt es Layouts mit CSS Grids zu realisieren?

Die beiden gängisten und beliebtesten Möglichkeiten zur Umsetzung von Layouts sind die CSS Eigenschaften Flexbox und Grid. Sie wurden entwickelt um Konstruktionen von flexiblen und responsiven Layouts zu ermöglichen.

Wie funktioniert das?

Mit dem Hinzufügen des Attributes zu einem Container werden automatisch alle Child-Elemente des Containers auf erster Ebene zu Flex-items (flex) oder Inhaltscontaintern (grid). Diese können dann mit weiteren Eigenschaften versehen werden.

Allgemeines zu Flexbox

Flexbox ist ein eindimensionales Layoutmodell und ermöglicht eine Platzverteilung zwischen Elementen.

Bei der Arbeit mit Flexbox muss man in zwei Achsen denken - der Hauptachse und der Querachse. Die Hauptachse wird durch die Eigenschaft flex-direction definiert, und die Querachse verläuft senkrecht dazu. Alles, was wir mit flexbox machen, bezieht sich auf diese Achsen.

flex-direction: row (default)

flex-direction: row | row-reverse | column | column-reverse

Ausrichtung

Ausrichtung auf der Hauptachse

justify-content: flex-start | center | flex-end | space-around | space-between

Es werden alle flex-items auf der Hauptachse entsprechend der jeweiligen Eigenschaft in dem freien Raum ausgerichtet.

Ausrichtung auf der Kreuzachse

align-items: flex-start | center | flex-end | baseline | stretch

Standardmäßig passt sich die Höhe oder Breite auf der Kreuzachse immer dem größten Element an (stretch). Wir können auch hier aber unterschiedliche Eigenschaften verwenden

Mehrspaltigkeit über das Container-Element

flex-wrap: wrap | no-wrap

Mit der Container-Eigenschaft flex-wrap kann festgelegt werden, dass Items in mehrere Spalten und Zeilen umbrechen.

Allgemeines zu CSS Grid

Flexbox ist ein eindimensionales Layoutmodell und ermöglicht eine Platzverteilung zwischen Elementen.

Flexbox ist eindimensional. Die Elemente werden im Prinzip nur auf einer Achse verschoben. (Hauptachse) Ein CSS Grid Layout bietet zwei Dimensionen zur Platzierung und Skalierung der Container. Statt nur auf einer Achse kann man mit CSS Grid ein Raster mit Zeilen und Spalten verwenden.

.grid {
   grid-template-columns: 2fr 1fr 1fr;
}

.grid {
   grid-template-rows: 2fr 1fr 1fr;
}

.grid {
   grid-template-columns: 2fr 1fr 1fr;
   grid-template-rows: 2fr 1fr 1fr;
}

.a {
   grid-row-start: 2;
   grid-column-start: 2;
}
